読み込み

csvファイルを読み込む際は、メニュー画面のFileからOpenを選択し、読み込みたいcsvファイルを選択してください。
読み込む際は設定ファイルを必ず用意してください。

csvファイルのフォーマット

  • 読み込むcsvファイルの1行目は必ずヘッダ行としてください。
  • ヘッダーに示された列数以下/以上の行が存在しないようにしてください。
  • 区切り文字はコンマ","としてください。
  • 文字コードはUTF-8として保存してください。

設定ファイル

本ソフトウェアはcsvファイルを読み込む際、同時にcsvファイルと同じフォルダにあるnumerous.settings.jsonファイルを自動的に読み込みます。
以下に示すフォーマットに従ってjsonファイルを用意してください。必須ではない項目はデフォルト値が読み込まれます。 

        {
            "name": "Name",                                     設定の名称(必須)
            "imagePathFormat": "${CSV_DIR}images/$[id].jpg",    読み込む画像の絶対パスとなるフォーマット(必須)詳細
            "columns": [                                        csvファイルの列の設定、以下の設定を配列として記述、順番に上から表示(デフォルトは全ての列を元のヘッダ名で表示)
                {
                    "name": "column name",                      csvファイル上のヘッダ名(必須)
                    "displayName": "display name",              ヘッダ名のソフトウェア上での表示名(デフォルトはcsvファイル上のヘッダ名)
                    "display": "false"                          ソフトウェア上でこの列を表示するかどうか(trueまたはfalse、デフォルトはtrue)
                }
            ]　　　　　　　　　　　　　　　　　　　　　 
        }

画像ファイルのパスの設定

画像ファイルへのパスは、csvファイルから一意に定めることのできるパスである必要があります。
imagePathFormatキーでは、以下に示すリテラルが使用可能です。

  • ${CSV_DIR}
    読み込んだcsvファイルが存在するフォルダのパスを示します。
    例)C:\Users\user\desktop\csv.csvを読み込んだ場合、${CSV_DIR}はC:\Users\user\desktop\に変換されます。
  • $[ヘッダ名]
    ヘッダ名に対応した値を示します。
    例)csvファイルが以下のとき、
    $[id]はそれぞれ0, 1, 2に
    $[color]はred, blue, blueに
    $[shape]はtriangle, square, triangleに
    $[imagepath]はC:\images\triangle\0.jpg、C:\images\square\0.jpg、C:\images\triangle\1.jpgに変換されます。 
    id, color,  shape,      imagepath
0,  red,    triangle,   C:\images\triangle\0.jpg
1,  blue,   square      C:\images\square\0.jpg
2,  blue,   triangle    C:\images\triangle\1.jpg
